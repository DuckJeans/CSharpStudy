학습목표:클래스와 메소드

네임스페이스----------------------

네임스페이스는 하나의 공간을 만든다
이름을 지정하여 비어있는 바구니를 채워주는 역할을 한다.

네임스페이스엔 메소드와 변수를 이용하여 데이터에 접근을 하고 사용할 수 있는 집합체를 얘기한다.

public,private--------------------------

public 은 공공의 라는 의미로 외부에서 전부 사용할 수 있게 하라는 의미이고
private는 class 내부에서만 사용하라는 접근 지정자를 명시하지 않으면 자동으로 private로 선언이 된다.

메소드---------------------------

메소드는 코드를 하나의 이름 아래 묶는 것을 의미합니다.
묶인 코드를 이름을 불러주는 것만으로 실행할 수 있습니다.
메소드를 호출하는 것을 Call method라고 한다.

접근 지정자 + 반환값 + 메소드의 이름 + () + {}

예)public void Show() {묶인 코드 내용들}

접근 지정자는 public 또는 private로 선언
반환값이 없는 경우에는 void
있는 경우에는 Int, char, string, float으로 반환
메소드의 이름 Show();
Show(이름);
3개의 기호가 합쳐져 Show메소드 안에 있는 코드를 호출하는 것이다.

말 그대로 내가 만들었던 로봇을 리모콘으로 명령하는 것과 같이
마트에서 장보듯이 아무것도 없던 장바구니에 코드들을 입력하여 꽉 채워넣은 후
그것을 계산하고 사용하는 것이라고 보면 쉽다.

반환값이 있는 함수인 경우에는 return + 변수의 이름 + ; 으로 해당하는 변수 값을 변환한다.
